Cách dùng

Cài xong, dùng ngay.

Không cần đăng ký, không cần API key, không cần server.

01
Cài từ Chrome Web Store
Bấm "Thêm vào Chrome", xác nhận quyền hạn. Biểu tượng ⚡ xuất hiện trên toolbar.
02
Mở popup
Click vào biểu tượng ⚡ trên bất kỳ trang nào. Chọn tab tương ứng với việc cần làm.
03
(Tuỳ chọn) Cấu hình Telegram
Nhập Bot Token và Chat ID nếu muốn gửi recording hoặc bug report qua Telegram.
04
Debug nhanh hơn
API Tester, recorder, bug report — tất cả trong tầm tay, không cần rời khỏi trang.
